# Catch the Star

## Функціонал

* **Анімований політ:** Зірка плавно рухається по діагоналі екрана.
* **Інтерактивна пауза:** Коли ви наводите курсор на зірку, вона зупиняється, щоб її було легше натиснути.
* **Зміна кольору:** При кліку на зірку вона випадковим чином змінює свій колір.
* **Адаптивність:** Проєкт працює у будь-якому сучасному браузері.

## Технології

* **HTML5:** Структура сторінки.
* **CSS3:** Малювання зірки через `clip-path`, анімація `@keyframes`.
* **Vanilla JavaScript:** Логіка обробки кліків та генерація випадкових кольорів.
